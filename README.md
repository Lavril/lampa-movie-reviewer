# Плагин Lampa для анализа отзывов на фильмы или сериалы

![Lampa](https://img.shields.io/badge/Lampa-Plugin-blue?style=flat-square)

Плагин для медиацентра [Lampa](http://lampa.mx/), который анализирует отзывы зрителей с TMDB при помощи нейросетей и выдает структурированную, краткую и понятную выжимку по любому фильму или сериалу.

Больше не нужно читать десятки однотипных рецензий — плагин сделает это за вас и покажет главные плюсы, минусы и кому стоит посмотреть данную картину.

## Возможности

- **Мнение зрителей:** Краткое резюме общих впечатлений аудитории.
- **Оценки критиков:** Глубокий анализ сюжета и технических аспектов.
- **Плюсы и минусы:** Четкие списки того, за что фильм хвалят и ругают.
- **Кому зайдет:** Определение целевой аудитории.

---

## Установка плагина в Lampa

Чтобы добавить этот плагин в ваш интерфейс Lampa, используйте ссылку на GitHub Pages:

1. Откройте Lampa.
2. Перейдите в **Настройки** ➔ **Расширения**.
3. Нажмите **Добавить плагин** (или введите URL).
4. Вставьте следующую ссылку:
   ```text
   https://lavril.github.io/lampa-movie-reviewer/plugin.js
   ```
5. Перезагрузите Lampa. В карточке любого фильма или сериала появится новая кнопка «Анализ».
